# PrimeStoreUI

## Project Documentation

### Overview
PrimeStoreUI is a user-friendly interface for managing and showcasing products in an online store.

### Features
- User authentication
- Product management
- Order processing

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Des-dee/PrimeStoreUI.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```

### Usage
- Navigate to the homepage to view products.
- Use the admin panel to manage inventory and orders.
